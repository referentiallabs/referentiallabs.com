Training for Functional Programming (using Haskell, PureScript, TypeScript, and Nix)

January 1, 0001 Â· 2 minutes read

Referential Labs, LLC offers functional programming training for both application and infrastructure software development staff.

Our courses are delivered online via the following mechanisms:

  • synchronous live video/audio check-ins (scheduled times)

  • recorded videos and screencasts

  • shared access to Git repositories containing sample code

  • shared access to VMs, Docker images, Amazon Machine Images, etc for development configuration

  • shared access to proprietary learning and reference material in PDF and HTML formats (dead tree versions are available upon request)

We can custom design our course content for your team's goals and also periodically offer individual professional developers the opportunity to learn new things via our public training offers on specific topics.

Recent open-to-public courses offerings included:

  • Continuous Delivery Pipelines with Nix

  • Getting started with Serverless (TypeScript edition)

Coming soon:

  • Getting started with Serverless (PureScript edition)

  • Building REST APIs in Haskell
